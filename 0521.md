#### 6091 함께 문제 푸는날
같은 날 동시에 가입한 3명의 사람들이 온라인 채점시스템에 들어와 문제를 푸는 날짜가 매우 규칙적이라고 할 때, 다시 모두 함께 문제를 풀게 되는 그날은 언제일까?<br>
각각 3일마다, 7일마다, 9일마다 한번씩 들어온다면?
```py
a, b, c = map(int, input().split())
d = 1

while d % a != 0 or d % b != 0 or d % c != 0:
  d = d +1

print(d)
```
