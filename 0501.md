#### 6001 출력하기 01
python에서 가장 기본적인 명령 >> 출력문<br>
print() 를 이용해 출력
```py
print("Hello")
```
* <details>출력값
  ```py
  Hello
  ```
  </details>
#### 6002 출력하기 02
공백()을 포함한 출력문<br>
print("문장1 문장2") or print("문장1", "문장2")
```py
print("Hello World")
print("Hello", "world")
```
* 출력값
  ```py
  Hello World
  ```
#### 6003 출력하기 03
줄바꿈 출력문<br>
Hello<br>
World<br>
두 줄에 걸쳐서 출력
```py
print("Hello")
print("World")
```
* 출력값
  ```py
  Hello
  World
  ```
#### 6004 출력하기 04
''(single quotation mark)작은 따옴표가 들어있는 출력문<br>
print("''")
```py
print('Hello')
```
* 출력값
  ```py
  'Hello'
  ```
#### 6005 출력하기 05
""(double quotation mark)큰 따옴표가 들어있는 출력문<br>
print('""')
```py
print('"Hello World"')
```
* 출력값
  ```py
  "Hello World"
  ```
